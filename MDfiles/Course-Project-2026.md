Database Systems 10127
Database Project

1.  Design the database
    a.  Prepare an ERD for the database. Your ERD should have at least 5 entities.
    b.  Create a schema for the tables.
    c.  Normalize the schema to 3NF.
    d.  Implement tables with primary keys, foreign keys and possibly other constraints.

2.  Create and fill the database
    a.  Create the database with CREATE commands in the database (not through the Java program). Make sure you have all the constraints and keys defined.
    b.  Fill the database with data using either INSERT commands or import from a file.

3.  Work with the database through your Java program
    a.  Update your Java program to use JDBC for all database access.
    b.  Support insert, update, delete, and search operations through the Java program.
    c.  Write at least 10 meaningful SQL queries
    d.  (bonus) Add triggers to your database. Be ready to explain why you chose the trigger that you implemented.

4.  A 5-8 minute presentation of your project, given in class
    a.  Presentation in English (described below).
    b.  Demonstration of running the program.
