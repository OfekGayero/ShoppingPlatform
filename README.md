E-Commerce Platform

 A computerized e-commerce project that simulates trading between buyers and sellers, similar to eBay. Users register to the system using a username and password.
User Roles & Functionality

    Sellers: Sellers add items they offer for sale. Every item added by a seller is logged into that specific seller's sales list in the store.

        Item Details: The offered items include an item name, price, a serial number (which automatically updates starting from 0 and increments by 1 for each new item in the store), and a product category chosen from 5 types: Kids, Electronics, Office, Clothing, and Other.

    Buyers: During registration, in addition to the basic user details, buyers enter a residential address where their products will be shipped at the end of a purchase.

The Purchasing Process

    Shopping Carts: Every purchase is managed via a shopping cart. When a buyer wants to purchase products, they must enter the name of the seller they wish to buy from and select the item name to add it to their cart.

    Checkout & Order History: After the buyer completes the purchase, the total cost of the order is displayed. The cart is then saved to their cart history so they can view past purchases. If they wish to reorder the exact same cart, they can easily copy it over to their current shopping cart.

System & Management Capabilities

The system also provides several administrative and tracking features, allowing you to:

    Print the list of registered buyers.

    Print the list of registered sellers.

    Update the products that each seller offers for sale.

    View and run queries on the information stored within the system.

Project was implemented in java with postgres sql integration.
